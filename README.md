# 🎬 Movie Chat Assistant

An interactive movie recommendation system powered by FastAPI, ChromaDB, and Streamlit. Get movie suggestions based on your mood, favorite actors, genres, and even sentiment analysis.

## 💡 Features

- Natural language movie recommendations  
- Sentiment-aware suggestions based on your preferences  
- FastAPI backend with LLM-assisted query processing  
- Vector similarity search using ChromaDB  
- Clean, responsive Streamlit UI  

---

## 🛠️ Tech Stack

- **Python**, **FastAPI**, **Streamlit**  
- **ChromaDB** for vector search  
- **LLM-based query cleaning**  
- Basic sentiment analysis integration  

---

## 📂 How to Run

1. Clone the repository  
2. Install dependencies  
3. Start the FastAPI backend  
4. Run the Streamlit frontend
